# CipherOwl Homebrew Tap

Official [Homebrew](https://brew.sh) tap for CipherOwl's own command-line tools and applications.

## Install

```sh
brew install cipherowl-ai/tap/<formula>
```

The first install automatically taps this repository. Equivalently:

```sh
brew tap cipherowl-ai/tap
brew install <formula>
```

Upgrade to the latest release at any time:

```sh
brew upgrade <formula>
```

## Available formulae

| Formula | Description |
| --- | --- |
| [`cipherowl-sr3`](Formula/cipherowl-sr3.rb) | CipherOwl SR3 CLI — address screening, risk reasoning, and reporting |
